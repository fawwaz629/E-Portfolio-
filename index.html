01 — Career Exploration

Security-Focused
Software Developer

The role I am pursuing is that of a Security-Focused Software Developer — referred to in the modern industry as an AI/ML Security Engineer. It occupies the space between software engineering, machine learning, and cybersecurity, with the primary function of building automated systems that detect and respond to threats before they cause harm.

This differs from a traditional security analyst role in an important respect: rather than responding to incidents after the fact, a Security-Focused Developer engineers the detection infrastructure itself. In 2025, that infrastructure increasingly relies on machine learning models trained to recognise malicious patterns in real-time data.

My Final Year Project, PhishShield, is a direct application of this principle: a Random Forest model trained on labelled URL data, deployed via Flask as a live web application, and integrated into a Chrome browser extension for real-time blocking.

Career trajectory: Junior Security Developer → Security Engineer → ML Security Lead → Principal Security Architect. Anchored at each stage by certifications — Security+, CEH, and CISSP.
